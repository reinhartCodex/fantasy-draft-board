# Fantasy Draft Board

An Expo Go-compatible fantasy football draft tracker using ESPN's 2026 PPR Top 300 rankings.

## Test on iPhone

1. Open this repository in Expo Snack.
2. Install Expo Go from the App Store.
3. Tap **Open in Expo Go** from Snack.

Features include player search, position filters, My Team and Other Team selections, crossed-off players, roster groups by position, undo, reset, and persistent local storage.

Data source: https://g.espncdn.com/s/ffldraftkit/26/NFL26_CS_PPR300.pdf?adddata=2026CS_PPR300
